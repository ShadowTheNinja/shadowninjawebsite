01 Game Overview

Chase the Sausage! is a single-player 2D arcade game where you control a hungry dachshund racing around a grid to collect as many sausages as possible. The more you eat, the longer your dog grows — just like Snake — but cuter and meatier.

The game gets progressively harder as your dog grows longer and the grid fills up with your tail. One wrong turn into a wall or yourself and it's game over. How many sausages can YOU eat before it all goes wrong?

02 Game Goal

Your dog is very hungry. Guide it around the screen to eat every sausage you can find. Each sausage you eat makes your dog one segment longer. The real challenge is steering your long dachshund body without crashing — because one wrong move ends the run!

🎯 Grow your dachshund until it fills the whole screen to achieve the ultimate win!
✅ Win Condition
Grow long enough to fill the screen
❌ Lose Condition
Run into a wall or your own tail
🌭 Points
+1 point per sausage collected
💀 Lives
1 — no second chances!

03 Player Controls

All controls use the keyboard arrow keys. The dog keeps moving automatically — you just steer!

  • ↑ Up Arrow Turn the dog upward — moves toward the top of the screen
  • ↓ Down Arrow Turn the dog downward — moves toward the bottom of the screen
  • ← Left Arrow Turn the dog left — moves toward the left side of the screen
  • → Right Arrow Turn the dog right — moves toward the right side of the screen
  • Space Pause / Unpause the game
  • R Restart the game after a Game Over

⚠️ Note: You cannot reverse directly — the dog can't turn 180° in one move (e.g. going right then immediately left). Plan ahead!

04 Game Mechanics & Logic

Main Actions a player can take:

  • Steer the dog using arrow keys
  • Collect sausages to earn points and grow the dog
  • Avoid walls and the dog's own tail
  • Pause strategically to plan a tricky route

Game Variables:

Variable Starting Value What It Tracks
score 0 Total sausages collected — goes up by 1 each time
dogLength 3 segments How many body pieces the dog has — grows with each sausage
speed 1 tile/tick How fast the dog moves — can increase as score gets higher
health 1 One life only — hitting anything ends the game
isAlive true Set to false when the dog crashes — triggers Game Over screen
sausagePosition random Where the next sausage spawns — always in an empty cell

If / Then Rules (Game Logic):

  • IF the player presses an arrow key → THEN change the dog's direction
  • IF the dog's head touches a sausage → THEN score +1, dog grows by 1 segment, spawn new sausage
  • IF the dog's head hits a wall → THEN isAlive = false, show Game Over screen
  • IF the dog's head hits its own tail → THEN isAlive = false, show Game Over screen
  • IF score reaches the winning threshold → THEN show You Win! screen
  • IF player presses Space → THEN toggle pause (freeze all movement)

05 Game Loop

Every game "tick" (a fraction of a second), these steps run in order — over and over until the game ends:

  1. Read player input — did they press an arrow key or Space?
  2. Update the dog's direction if a valid key was pressed
  3. Move the dog one tile forward in the current direction
  4. Check if the dog's head hit a wall → Game Over if yes
  5. Check if the dog's head hit its own body → Game Over if yes
  6. Check if the dog's head is on the sausage → collect it, grow, spawn new sausage
  7. Update and display the score on screen
  8. Check if the win condition is met → show victory screen
  9. If no win or lose, wait for next tick and repeat

06 Playtesting & Changes

After letting real players try the game, we found these problems and made fixes:

⚠️ Problem Found

Players said the game was too easy — they could keep going forever without any challenge.

✅ Changes Made

Added more wall obstacles in the middle of the grid. Also increased the dog's movement speed as the score got higher, making long runs harder to control.

⚠️ Problem Found

Some players accidentally reversed their direction and instantly hit their own tail right at the start.

✅ Changes Made

Added a rule that prevents the dog from turning 180° in a single move. You can't go from moving right to immediately moving left — the input is ignored and you continue in your current direction.

⚠️ Problem Found

Players didn't know they had lost — the game just froze without any feedback.

✅ Changes Made

Added a clear "Game Over" overlay showing the player's final score and a message to press R to restart.

08 Designer Reflection

The hardest part of this project was making the game feel fun. It's easy to make a game that works, but we wanted players to actually enjoy it. We spent a lot of time tweaking how fast the dog moves and how quickly the speed increases — getting that balance right took many playtests.

We also struggled with collision detection at first. The code wasn't correctly checking if the dog's head overlapped its own tail, so players could phase through themselves. Fixing that bug taught us a lot about how loops and coordinate comparisons work in JavaScript.

Our game uses core programming concepts — variables (score, speed, length), if statements (collision checks, win/lose conditions), and loops (the game tick that runs every frame). Seeing these concepts come together to make something actually playable was the most rewarding part.

If we had more time, we would add: a high score leaderboard, different obstacle types, a second sausage that appears randomly for bonus points, and sound effects when you eat or crash.
